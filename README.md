## 🚧 | Prerequisites

- [Node.js 16+](https://nodejs.org/en/download/)

## 📝 | Tutorial

> The `settings.yml` file should be configured first.

Install all dependencies and deploy Slash Commands
```sh
npm install
```
Start the bot with start.bat
